# Diagnóstico de Problemas en el Servicio Psicopedagógico de la UPLA

## Desarrollo de una Página Web mediante la Metodología XP

## Descripción del proyecto

Este proyecto tiene como finalidad analizar los problemas presentes en el Servicio Psicopedagógico de la Facultad de Ingeniería de la Universidad Peruana Los Andes (UPLA) y proponer una solución tecnológica mediante el desarrollo de una plataforma web utilizando la metodología ágil Extreme Programming (XP).

Durante el desarrollo se identificaron diversas dificultades relacionadas con el proceso de agendamiento de citas, la gestión manual de la información, la comunicación con los estudiantes y el manejo de expedientes. A partir de este análisis se diseñó e implementó una propuesta que mejora la organización, la eficiencia y la accesibilidad del servicio.

## Objetivos

- Analizar la situación actual del Servicio Psicopedagógico.
- Identificar las principales problemáticas del proceso de atención.
- Justificar la aplicación de la metodología XP como solución.
- Diseñar y desarrollar una plataforma web para la gestión de citas.
- Implementar historias de usuario, casos de prueba e iteraciones siguiendo las prácticas de XP.

## Características del proyecto

- Análisis de la problemática.
- Aplicación de la metodología XP.
- Desarrollo basado en historias de usuario.
- Planificación por iteraciones.
- Casos de prueba y validación.
- Diseño de una plataforma web para la gestión de citas.
- Documentación completa del proceso de desarrollo.

## Conclusión

La aplicación de la metodología XP permitió desarrollar una solución enfocada en mejorar el proceso de agendamiento de citas del Servicio Psicopedagógico de la UPLA mediante un desarrollo iterativo, flexible y centrado en las necesidades del usuario. La plataforma propuesta contribuye a optimizar la gestión de la información, reducir los tiempos de atención y brindar un servicio más eficiente a la comunidad universitaria.
